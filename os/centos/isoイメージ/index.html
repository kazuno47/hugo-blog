01.ISOイメージ

ISO イメージの種類

# image note
1 DVD ISO 標準パッケージ入りディスクイメージ。開発や自宅で使う場合などはこれかな。
2 Everything ISO 標準パッケージ以外に、追加でパッケージが入っているディスクイメージ。ローカルにリポジトリを立てる場合に使ったりするらしい。
3 Minimal ISO 必要最低限のパッケージだけが含まれているディスクイメージ。業務の本番環境ではこれに、構成管理ツールで追加設定するといいかも。
